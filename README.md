# Оптимизация инвестиционного портфеля с использованием динамического программирования

Учебный проект по решению задачи оптимизации инвестиций с помощью одномерного и двумерного динамического программирования.

## Задачи
- Максимизировать прибыль от акций при ограниченном бюджете (аналог задачи о рюкзаке).
- Учесть ограничение на риск: не более D% средств в высокорисковые активы (акции).
- Сравнить с жадным алгоритмом и визуализировать зависимость прибыли от бюджета.

## Структура проекта
- `investment_optimizer.py` — класс `InvestmentOptimizer` с методами `max_profit_1d` и `max_profit_2d`
- `main.py` — пример запуска и тестовые данные
- `visualize.py` — построение графика прибыли от бюджета
